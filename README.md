# 🎭 Versos Vivos — Poetry Web Platform

> *"No escribo para ser leído, escribo para que recuerdes que estás vivo."*

Web platform for marketing and community building around live poetry events.
Built for poets, presenters and creators who want to connect with their audience.

---

## 🌐 Live Demo
> Coming soon — [versosvivos.netlify.app](#)

---

## 📋 Features

- 🏠 **Landing Page** — Elegant hero section with animated particles, artist bio and past events gallery
- 📅 **Events Page** — Interactive calendar, event registration modal and audience analytics dashboard
- 💬 **Community Page** — Testimonial wall with emotion tags and filtering system
- 📱 **Social integration** — Instagram, TikTok, YouTube, Twitter/X
- 📊 **Analytics** — Attendance tracking, interest tags, audience origin charts

## 🚀 Getting Started

No installation required. Pure HTML/CSS/JS project.

```bash
# Clone the repository
git clone https://github.com/sit0dwad3-art/Poetry_Web.git

# Open in browser
# Simply open index.html in your browser
# Or use Live Server in VS Code (recommended)
```
